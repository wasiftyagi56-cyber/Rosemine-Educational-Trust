# Rosemine Educational Trust — Official Website

A professional, fully responsive website built for an educational trust.

**Live Demo:** [rosemine-educational-trust-m7dr.vercel.app](https://rosemine-educational-trust-m7dr.vercel.app)

---

## Features

- About the trust — mission, vision, and values
- Programs and courses section
- Team and faculty profiles
- News and events section
- Contact page with enquiry form
- Fully responsive on all devices

---

## Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure and Semantics |
| CSS3 | Styling and Responsive Layout |
| JavaScript | Interactivity |
| Vercel | Deployment and Hosting |
